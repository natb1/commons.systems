# Writing Style

## Use simple, direct language. Avoid corporate jargon.
- Choose specific verbs over buzzwords
- Use technical language for technical concepts, not business metaphors
- When in doubt, pick the simpler phrase

Replace These Phrases
- **"leverages"** → use "uses"
- **"synergy"** → be specific about the collaboration
- **"paradigm shift"** → describe the actual change
- **"circle back"** → use "follow up" or "discuss later"
- **"move the needle"** → use "make progress" or "improve"
- **"low-hanging fruit"** → use "quick wins" or "easy improvements"
- **"deep dive"** → use "detailed analysis" or "thorough review"
- **"touch base"** → use "check in" or "discuss"
- **"think outside the box"** → use "be creative" or "find new approaches"
- **"at the end of the day"** → use "ultimately" or "finally"
- **"going forward"** → "in the future" or omit
- **"bandwidth"** (for time) → "time" or "capacity"
- **"on my radar"** → use "aware of" or "tracking"
- **"take this offline"** → use "discuss separately"
