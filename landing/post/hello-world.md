# Hello World

Welcome to commons.systems — a blog about agentic coding workflows and the tools we build to make them work.

This is the first post. The site itself was built using the agentic workflow it describes, which feels like an appropriate way to start.

## What This Is

commons.systems is a monorepo for experimenting with AI-assisted software development. The landing site you're reading now was scaffolded using a Go-based tool, configured with Firebase for hosting and Firestore for data, and built with a minimal TypeScript frontend — no framework, just the platform.
